🎮 Game Review: Ghost of Tsushima

⬅ Back to Blog

📚 Developer: Sucker Punch Studios

🎭 Genre: Action/Adventure, Open World, Chink

⏳ Release Date: 17th July 2020

⭐ Rating: 9/10

This game is absolutely gorgeous playing this at home 4K with the surround sound system was absolutely amazing. The game got infinitely better when I set it to Lethal. Where you have low heath and the enemies also the parrying is really hard but is so rewarding.
